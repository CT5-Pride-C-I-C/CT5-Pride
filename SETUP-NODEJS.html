This project requires Node.js to run the local development server and handle Git operations.

Step 1: Install Node.js

Download and install Node.js from the official website:

Download Node.js

Choose the LTS version for best compatibility and stability.

Step 2: Verify Installation

After installing Node.js, open a command prompt or PowerShell and run:

node --version npm --version

You should see version numbers displayed for both commands.

Step 3: Setup Project

Once Node.js is installed, you can set up the project:

# Navigate to project directory cd CT5-Pride # Install dependencies npm install # Setup environment (GitHub token) npm run setup # Start the server npm start

Step 4: Access the Website

After starting the server, you can access:

  • Main Website: http://localhost:3000
  • Admin Panel: http://localhost:3000/admin-add-role.html
